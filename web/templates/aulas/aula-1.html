Aula 1 - Tecnologias e Ferramentas - Angular

Nesta aula será apresentado as tecnologias e ferramentas que utilizaremos durante a Unidade 2.

Objetivos de aprendizagem

Os objetivos da aula são:

  • Conceituar "Single Page Application (SPA)" Aplicação de página única;
  • Entender a estrutura de pastas de uma aplicação angular 8;
  • Criar o esqueleto de uma aplicação angular 8;

O que fazer?

Para atingir os objetivos citados acima, você deve acessar todos os conteúdos listados abaixo, na ordem em que foram dispostos.
Ao final da aula, você deverá ter conseguido executar as seguintes tarefas em seu computador:

  • Instalar o Node.js e o NPM
  • Instalar o Angular CLI
  • Criar uma aplicação Angular 8 usando o Angular CLI
  • Compreender a estrutura de pastas de uma aplicação Angular CLI
  • Executar a aplicação e acessar o navegador
